# AMORA - Beauty Platform

AMORA is a scalable beauty ecosystem where users can:
- Watch beauty videos (like TikTok/Instagram Reels)
- Learn beauty skills
- Follow creators and salons
- Book salon services (future)
- Buy beauty products (future)
- Use AI beauty assistant (future)

---

# 1. TECH STACK (LOCKED)

Frontend:
- Next.js

Backend:
- NestJS

Authentication:
- Firebase Authentication (Google + OTP)

Database:
- Supabase PostgreSQL

ORM:
- Prisma

Media Storage:
- Cloudinary

Auth Strategy:
- Firebase Auth → NestJS verification → JWT system

---

# 2. CORE ARCHITECTURE

User Flow:

1. User logs in via Firebase (Google / OTP)
2. Firebase returns ID Token
3. NestJS verifies token
4. If user not exists → create in PostgreSQL
5. JWT generated by NestJS
6. JWT used for all API requests

---

# 3. ROLE SYSTEM

Roles:
- customer
- creator
- salon
- admin

One user can have multiple roles.
